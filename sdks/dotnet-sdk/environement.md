# Environement {#development-environement}

---

* IDE
  * Visual Studio 2013 above version
* Runtime
  * .Net Framework 4.5.2
* Nuget Package
  * [WISEPaaS.SCADA.DotNet.SDK](https://www.nuget.org/packages/WISEPaaS.SCADA.DotNet.SDK)
